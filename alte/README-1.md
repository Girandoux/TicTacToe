# Tic-Tac-Toe II v3

## Projektbeschreibung

Tic-Tac-Toe II v3 ist eine moderne Desktop-Version des klassischen Tic-Tac-Toe-Spiels, entwickelt mit Python und Tkinter.

Das Projekt wurde modular aufgebaut und enthält eine grafische Benutzeroberfläche (GUI), verschiedene Schwierigkeitsgrade sowie ein Punktesystem.

Der Spieler tritt gegen eine einfache künstliche Intelligenz (KI) an.

---

# Funktionen

- Modernes GUI-Fenster mit Tkinter
- Spieler gegen Computer
- Drei Schwierigkeitsgrade
- Punktestand-System
- Restart-Button
- Spielername im Fenster
- Modulare Projektstruktur
- Kommentierter und übersichtlicher Code

---

# Schwierigkeitsgrade

## Einfach
Der Computer wählt zufällige Züge.

## Mittel
Der Computer blockiert mögliche Gewinnzüge des Spielers.

## Schwer
Der Computer versucht selbst zu gewinnen und blockiert zusätzlich den Spieler strategisch.
